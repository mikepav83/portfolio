ET-712 Web Client Programming

Javascript and JQuery

In the gallery below, you will find samples of my work done for the ET-712 class.

"Tip Calculator"
This is a simple tip calculator. Similar to ones you find in a phone app. It calculates a fixed percentage. Will have to work on making a more advanced one where the percentage is entirely customizable. Had to style it with a delicious burger of course. Looking at it will make you hungry.
"Lifetime Supply of Coffee Calculator"
Seeing as how coffee is my favorite beverage in the world, I just had to do a lifetime supply calculator based on it. After doing the math myself, I have a long way to go in terms of quantity of coffee for consumption goes. Styled with a background that resembles my daily routine.
"Fun With jQuery Animation"
This piece was a difficult but enjoyable one. The hardest part was animating the shape with colors. I wrote the code thinking it would just change, but it needed a plugin to work. Proof that there is always something new to learn as you go along. Enjoy the show.
"JS Snippet APIs"
Used JSON to get the data for the youtube videos in this one. I have never heard of JSON or knew what it was used for until I did this assignment. It was interesting to do something like this even though I don't see much of a use for it in daily web development practices. That could change.
"Ajax/JSON API"
Using Ajax and JSON, I was able to get a random search result from youtube on five videos involving the sport of golf. You can change the search topic and quantity with some quick tweaks. I happen to like golf so that was my topic of choice. Also added the search bar so anyone could search for videos on youtube.
"Fortune Teller"
Created a fortune telling input form using user variables to create the outcome. This was my very first javascript assignments. I found it to be difficult at first but eventually pulled it off. The hardest part was styline something like this since my CSS knowledge at the time was very limited.
